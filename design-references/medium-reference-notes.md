## 可复刻为个人站功能

| 区域 | Medium 行为 | 个人站实现 |
|---|---|---|
| 顶部导航 | Logo、搜索、写作入口、账号入口 | 品牌、全站搜索、写文章、管理入口 |
| 搜索框 | 输入后进入搜索结果页，保留 query | 建独立搜索结果页，支持文章/标签切换 |
| 搜索结果 | Stories / People / Publications / Topics / Lists | 保留文章和标签，去掉社交类 tab |
| 中间内容区 | 文章卡片：来源、作者、标题、摘要、日期、互动、缩略图 | 文章卡片保留标题/摘要/日期/阅读时长/标签/封面 |
| 话题页 | 话题名、统计、推荐文章流 | 标签页：标签名、文章数量、相关文章 |
| 文章页 | 标题、作者、时间、阅读时长、操作栏、正文、底部作者/刊物信息 | 标题、作者、日期、阅读时长、分享、RSS、目录、相关文章 |
| 右侧栏 | 推荐、话题、作者/刊物信息 | 关于作者、精选文章、推荐标签、最近更新 |

## 当前访问限制

| 限制 | 说明 |
|---|---|
| 登录后首页不可访问 | Medium 的左侧 sidebar + 个性化 For You feed 需要登录状态，只能参考用户已提供截图 |
| 部分文章有会员墙 | 可参考顶部和正文开头，不能完整采集全文结构 |
| 搜索初次触发 Cloudflare | 等待后可进入结果页，但应避免依赖自动反复抓取 |
